# admindashboard
A responsive admin dashboard built using HTML, CSS, Bootstrap, and JavaScript. 
# This project is currently under development. New features and improvements will be added over time.
## Features
- Dashboard statistics (Revenue, Orders, Users)
- User management (Add, Edit, Delete)
- Pagination
- Activity feed with live updates
- Dark/Light theme toggle
- Data export to file
- Responsive card based layout

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- 
## Planned Improvements
- Local storage / data persistence
- Responsive sidebar for mobile devices
- Charts & analytics visualization
- Authentication simulation
- UI enhancements and animations

## Purpose
This project was created for learning and portfolio purposes to practice frontend development and dashboard UI design.
